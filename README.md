# Messaging App (Backend)

## Messaging App (Backend) - Overview

This is the eight project of the "NodeJS" course within "The Odin Project".

This project is used to practice creating fullstack application.

In particular, this project aims to create a messaging application.

*Note: This repository is for the API backend. This side of the application only includes the app's logic and data handling.*

For the client visit [this](https://github.com/Dur4nt3/messaging_project_front) repository.

For the overview repository, visit [this](https://github.com/Dur4nt3/messaging_project_overview) link.

## Skills Demonstrated

- React

- React Router

- RESTful API

- NodeJS

- Express

- PassportJS (JWT)

- PostgreSQL

- Prisma ORM
